# Personal Portfolio

Minimal personal portfolio built using Jekyll and hosted on GitHub Pages.

The site showcases projects, professional background, and occasional writing.

## Sections
- Home: brief overview and featured projects
- Projects
- About: resume-style professional background
- Writing: short essays
- Resume: downloadable PDF

## Tech Stack
- Jekyll
- Markdown
- GitHub Pages

## Purpose
Designed to emphasize clarity and correctness over visual noise.
